# Beckett Pizza Plaza - 4-for-3 Offer Calculator

## Description
This Python program calculates the total cost of a pizza order from Beckett Pizza Plaza.  
It applies a "4-for-3" offer: the cheapest pizza is free when ordering four pizzas.  
The program supports both interactive input and reading orders from a file.

## How to Run

### Interactive Mode:
```bash
python pizza_order.py
