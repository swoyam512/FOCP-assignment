# Password Verification and File Processing Program

## Description
This Python program first verifies a user's password for security. The password must be at least 9 characters long, and the user must correctly provide 3 randomly selected letters from it. If the password check passes, the program reads a text file provided as a command-line argument and outputs the number of lines, words, and characters in the file.
